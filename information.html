C++ Setup



Before you start learning C++, it is very necessary to set up C++. We’ll be needed a two software’s to be installed in your computer.

Text Editor



To write a C++ program we’ll need a text editor for example we can try windows notepad, OS edit command, Epsilon or EMACS. It is necessary to know, what is source files? Source files are the created one files using text editor typically they are named with the extension of .cpp or .cp or .c. A text editor is where you’ll be writing all your code.

C++ Compiler



What is C++ Compiler? It is actually a compiler that compile your source code into final executable program. The widely used and also freely available that people used is GNU C/C++ compiler. If you want to try something else you can havef either HP or Solaris depending on which Operating System you are using.

Installing GNU C/C++ Compiler



UNIX/Linux Installation



If you are using linux or UNIX you make sure you have installed GCC which could be checked using this following command:

$ g++ -v

If you have GCC on your computer then this following message should be printed out:

Using built-in specs.

Target: 1386-redhat-linux

Configured with: ../configure –prefix=/usc……

Thread model” posix

Gcc version 4.1.2 20080704 (Red Hat 4.1.2-46)

If that messages is not showing up then GCC is not installed, then you will have to install it yourself.

Mac OS X Installation



If you use Mac OS X, the easiest way to obtain GCC is to download the Xcode development environment from Apple's website and follow the simple installation instructions.

Windows Installation



If you use Windows OS then install MinGW. To install MinGW you can go to MinGW homepage then follow the instruction to download that page. While installing MinGW, at a minimum, you must install gcc-core, gcc-g++, binutils, and the MinGW runtime, but you may wish to install more. Add the bin subdirectory of your MinGW installation to your PATH environment variable so that you can specify these tools on the command line by their simple names. After the installation is completed now you be able to run gcc or g++ or any other several GNU tools from Windows command line.
